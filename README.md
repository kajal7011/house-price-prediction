# House Price Prediction using Machine Learning

## Project Overview
This project predicts house prices using Machine Learning. 
The model analyzes different features such as area, number of bedrooms, 
and other factors to estimate the price of a house.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Machine Learning Process
1. Data Collection
2. Data Preprocessing
3. Feature Selection
4. Model Training
5. Model Evaluation

## Project Output
The model predicts house prices based on given input features and evaluates
performance using regression metrics.
